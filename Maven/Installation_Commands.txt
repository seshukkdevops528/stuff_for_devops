Maven Dir Structure
===================
/bin --> Contains all binary files {mvn etc...}
/boot --> Contains jar files used at onetime
/conf --> configuration file {setings.xml}
/lib --> Library Files {jar files}

Default xml file names for build tools
======================================
for Maven --> pom.xml {Project Object Model [POM]}
for Ant --> build.xml
for Gradle --> build.gradle

Maven repositories --> Maven repositories are locations where Maven artifacts (JAR files, plugins, and other project dependencies) are stored and can be retrieved from.
maven local repository  --> The local repository is a directory on your local machine where Maven stores downloaded artifacts. By default, it is located in the .m2 directory within the user's home directory.
    linux: ~/.m2/repository
    windows: C:\Users\prasanth\.m2\repository
    mac book: /users/prasanth/.m2/repository
maven central repository --> The central repository is the default repository used by Maven, hosted by the Maven community. It contains a vast number of commonly used libraries. If a dependency is not found in the local repository, Maven searches the central repository.
URL: https://repo.maven.apache.org/maven2/
